contextLabeler NVDA Add-on — Windows Setup Commands
=====================================================

PREREQUISITES
-------------
- NVDA installed (nvaccess.org)
- Git repo downloaded to Desktop as: nvda_reader-main
- Anthropic API key ready


STEP 1 — LINK THE ADD-ON INTO NVDA SCRATCHPAD
----------------------------------------------
Open PowerShell as Administrator (right-click PowerShell -> Run as administrator)
Then run:

    New-Item -ItemType SymbolicLink -Path "$env:APPDATA\nvda\scratchpad\globalPlugins\contextLabeler" -Target "$env:USERPROFILE\Desktop\nvda_reader-main\globalPlugins\contextLabeler"


STEP 2 — ENABLE SCRATCHPAD IN NVDA
------------------------------------
Do this in NVDA (not command line):

    1. Press NVDA+N to open NVDA menu
    2. Go to Preferences -> Settings -> Advanced
    3. Check "Enable loading of custom code from the Developer Scratchpad Directory"
    4. Click OK
    5. Press NVDA+Ctrl+F3 to reload plugins


STEP 3 — SET YOUR ANTHROPIC API KEY
-------------------------------------
Do this in NVDA (not command line):

    1. Press NVDA+N to open NVDA menu
    2. Go to Preferences -> Settings -> Context Labeler
    3. Paste your Anthropic API key
    4. Click OK


STEP 4 — TEST IT
-----------------
    1. Open this file in Chrome or Edge:
       Desktop\nvda_reader-main\tests\test-bad-a11y.html

    2. Press Tab a few times to land on an unlabeled button

    3. Press NVDA+Shift+L

    4. NVDA should speak a label within ~2 seconds


AFTER ANY CODE CHANGE (reload without restarting NVDA)
-------------------------------------------------------
    Press NVDA+Ctrl+F3


TROUBLESHOOTING — VIEW NVDA LOG
---------------------------------
    Press NVDA+F1, then click "View Log"
    Look for lines starting with ERROR


USEFUL NVDA SHORTCUTS
----------------------
    NVDA+N          Open NVDA menu
    NVDA+Shift+L    Run the contextLabeler (our hotkey)
    NVDA+Ctrl+F3    Reload all plugins
    NVDA+F1         Open NVDA help / log viewer
    NVDA+Q          Quit NVDA
