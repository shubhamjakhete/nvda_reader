# nvda_reader — Context Labeler for NVDA

An NVDA add-on that uses an ontology + Claude AI to fill in missing accessibility labels for unlabeled UI elements (icon buttons, alt-less images, custom widgets).

When a blind user lands on an unlabeled button, they press `NVDA+Shift+L` and NVDA speaks a meaningful label, generated by Claude Haiku and validated against a hand-edited W3C RDF/Turtle ontology.

## Status

## Tech stack

- Python 3.11 (NVDA-bundled)
- NVDA 2024.1+ (Windows)
- RDFLib (vendored) for ontology + SPARQL
- RDF/Turtle for the ontology format
- Anthropic Claude Haiku for classification
